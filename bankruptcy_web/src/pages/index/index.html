关于我们

ABOUT US

中心简介 MORE

中国人民大学破产法研究中心成立于2006年，是经中国人民大学批准设立，研究中心秉承教学、研究、实践三位一体的理念，致力于破产立法和司法的进步，研究破产法的基础理论问题，关注破产法理论的发展动向，推动中国破产法制的进步。

学会简介 MORE

北京市破产法学会成立于2010年，由中国人民大学法学院、首都经济贸易大学法学院、北京市炜衡律师事务所等单位自愿联合发起成立，是经北京市民政局核准登记的非营利性社团法人，业务主管单位是北京市法学会。

历届论坛 MORE

中国破产法论坛是中国人民大学破产法研究中心联合最高人民法院民二庭共同创办的破产法领域最高规格、最大规模、最为权威的品牌学术活动，自2008年至今，共举办了七届大规模的论坛和六次破产法专题研讨会。    

研究团队 MORE

我们团队涵盖理论界与实务界，形成了老中青相结合的人才梯队。我们将秉承“海纳百川，有容乃大”之发展理念，陆续聘请立法机关、司法机关、实务部门、高等院校以及国外破产法领域的专家学者加入研究团队，充实队伍。
